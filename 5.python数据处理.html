一、numpy基础¶

numpy是高效的数值计算和数据分析的基础包，其核心是称为ndarray的数据类型，一种多维数组，能够进行快速的算术和矩阵运算。

使用python标准库

In [75]:
import random
position = 0
walk = [position]
steps = 1000
for i in range(steps):
    step = 1 if random.randint(0, 1) else -1
    position += step
    walk.append(position)
